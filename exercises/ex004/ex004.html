Exercício 04 - Escrevendo Dados no HTML com JS

O que este código está fazendo?

close

Neste código, estou pegando o nome do usuário por prompt. Com este nome dentro de uma variável, eu insiro informações dentro do HTML com o método .insertAdjacentHTML() de uma div ou elemento qualquer. Usando este método juntamente com o parâmetro beforeend, conseguimos injetar várias linhas de HTML neste elemento. No caso, eu insiro várias template strings atribuindo o nome do usuário com modificações ou demonstrando estatísticas dele. Isso foi feito utilizando o atributo .length e os métodos .toUpperCase() e .toLowerCase().
