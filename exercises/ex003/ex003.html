Aqui, estou lendo dois números do usuário via prompt. Em seguida, estou utilizando o método parseInt(n) do tipo Number, para converter as respostas do usuário em números inteiros. Finalmente, esses valores convertidos são somados e esta soma armazenada na variável soma.

Para mostrar o resultado, estou usando uma template string para mostrar o resultado para o usuário através de um alert. Template strings usam placeholders para apresentar as variáveis, sendo desnecessário usar concatenação.
