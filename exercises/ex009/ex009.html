O que este código está fazendo?

close

Neste código, utilizamos condições aninhadas para mostrar o status de um aluno na tela. A lógica se baseia em verificar o intervalo que a nota do usuário se encontra, para mostrar se ele está Aprovado, Reprovado ou em Recuperação.

A função invocada pelo click no botão inicialmente verifica a existência da nota no input. Caso afirmativo, o código então verifica se a nota é menor que 5.0 (reprovado); menor que 7.0 (recuperação); ou, por fim, maior que 7.0 (aprovado).
